## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Visit `http://localhost:5173` to see the website.

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Sections

1. **Hero** - Animated hero section with floating orbs and gradient effects
2. **Services** - Showcase of services with hover animations
3. **About** - Company information with statistics
4. **Portfolio** - Project showcase with interactive cards
5. **Contact** - Contact form and information
6. **Footer** - Footer with links and social media

## Customization

You can customize colors, animations, and content by editing:
- `tailwind.config.js` - Theme and color customization
- Component files in `src/components/` - Content and layout
- `src/index.css` - Global styles and utilities
